

# HTML & CSS Practice Projects 🌎

> A collection of hands-on HTML and CSS projects built while following the **HTML & CSS Full Course** by Bro Code.

This repository contains all the practice projects, layouts, and code examples from the [HTML & CSS Full Course](https://youtu.be/HGTJBPNC-Gw) — a comprehensive 4+ hour tutorial covering everything from basic HTML structure to advanced CSS styling, layouts, and animations.

---

## 📚 Course Overview

The course is structured into **37 chapters**, each focusing on a specific HTML or CSS concept. Every chapter includes practical coding exercises and mini-projects that reinforce the learning. This repo serves as my personal workbook and a reference for anyone learning HTML and CSS alongside the video.

## 🛠️ How to Run

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/your-repo-name.git
   ```

2. **Navigate to the project folder**
   ```bash
   cd your-repo-name
   ```

3. **Open any project**
   - Each topic is in its own folder (e.g., `/forms`, `/flexbox`, `/animations`, etc.)
   - Open the `index.html` file in your browser
   - Or use the **Live Server** extension in VS Code for auto-refresh

## 🎯 Learning Goals

- Build a strong foundation in HTML structure and semantics
- Master CSS styling (colors, fonts, borders, shadows, spacing)
- Understand layout techniques (float, display, position, flexbox)
- Learn modern CSS (transforms, animations, pseudo-classes, pseudo-elements)
- Build complete, responsive-looking web pages
- Gain confidence to style real-world websites
